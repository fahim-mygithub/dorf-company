House rulesbreak these and the card will not read

1. Mark is the only multiplier. 🎯 +25% (and enemy-side 💥 Vulnerable ×1.5). Everything else is flat, or the numbers stop being legible.
2. Resolution order is fixed: base → +flat (Channel, Aura, Whetstone) → ×Mark → block → hp → Retaliate. Write the card so it reads correctly in that order.
3. Cost is the honest brake. 0 = tempo, 1 = the norm, 2 = a real commitment. There is no 3.
4. Live numbers on the face. Db.describe() renders the body — if your card's value changes with state, it must show the current value, in green, on the card.
5. Neutral must be class-blind. If it only makes sense on one dwarf, it belongs in CLASS_REWARDS, not REWARD_POOL.
